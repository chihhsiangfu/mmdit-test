## Project goals

- This is a project focused on developing and learning `MMDiT`.
- Uses `Rectified Flow`.
- After opening the project, prioritize reading all files in the `mmdit-motion` project and understand how to perform the following operations on it:
  - training
  - resuming training
  - sampling
  - summarizing model info
  - gradio app demo
  - ipynb entry for colab

## Development requirements

- Ensure the training script can train on cuda / mps / cpu hardware.
- Ensure the training script has optimizations specifically for H100.
- Ensure the script always supports:
  - training
  - resuming training
  - sampling
  - summarizing model info
  - gradio app demo
  - ipynb entry for colab
